## Command Using 
```sh
  # Check a single account
  python checker.py check --username tiktok
  
  # Check multiple accounts from file
  python checker.py check --input accounts.txt --output results.json --format json
  
  # Check with proxy rotation
  python checker.py check --username tiktok --proxies proxies.txt
  
  # Bulk verify with 10 threads
  python checker.py check --input accounts.txt --threads 10 --output results.csv
  
  # Check username availability
  python checker.py available --username desiredname
  
  # Scan for available usernames from a wordlist
  python checker.py scan --wordlist words.txt --available-only --output available.txt
```
